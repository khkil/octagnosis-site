## 개발 환경 설정

# 패키지 설치

yarn

# 프로젝트 실행

yarn start

# 프로젝트 빌드

yarn build

## url 설계

## - 보안

1. 로그인 : /auth/login
2. 회원가입 : /auth/sign-up

## - 검사

1. 시작 페이지 : /inpections/{inspectionIdx}/pages/start
2. 검사 페이지 : /inpections/{inspectionIdx}/pages/{page}
3. 완료 페이지 : /inpections/{inspectionIdx}/pages/result
